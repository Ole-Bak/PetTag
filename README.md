# PetTag

A comprehensive Fullstack application designed for managing pet profiles and tags.

## 🚀 Tech Stack

### Frontend

- **Framework:** Angular
- **Language:** TypeScript
- **Styling:** Tailwind CSS, SCSS, HTML

### Backend

- **Framework:** NestJS
- **Language:** TypeScript

### Infrastructure & DevOps

- **Containerization:** Docker, Docker Compose

---

## 📋 Features

- Pet profile management
- Secure data handling
- Responsive design with Tailwind CSS
- Fullstack integration between Angular and NestJS
